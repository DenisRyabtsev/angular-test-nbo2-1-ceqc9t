Исходные данные:

  • JsonService содержит метод getData для получения данных и pushData для их условной отправки.
  • getData возвращает JSON с данными измерения счетчика энергии за две недели
  • Данные в виде пары timestamp (ms) и потребленной энергии за последние 15 минут (файл расположен ./assets/test.json)

Задание:

Изменить тестовый проект, чтобы вместо текста задания выполнял следующие пункты:

Перед выполнением каждого пункта сделать оценку сколько он может занять времени. Засечь время и записать сколько времени понадобилось по факту.
  • Получить данные getData.
  • Вывести название счетчика meterName (см исходные данные)
  • Вывести название канала данных channelName (см исходные данные)
  • Показать кнопку, по нажатию которой находится час, когда было использовано максимум электричества (ответ отпраляется в метод pushData)
  • Показать кнопку, по нажатию которой находится день, когда было использовано минимум электричества (ответ отпраляется в метод pushData)
  • Показать кнопку, по нажатию которой вместо этой кнопки будет показана среднечасовая использованная энергия
